Forward Propagation

Standard Layer (without Batch Normalization)

For layer lll:

Z[l]=W[l]A[l−1]+b[l]Z^{[l]} = W^{[l]} A^{[l-1]} + b^{[l]} Z[l]=W[l]A[l−1]+b[l]

Shapes: (n[l],m)=(n[l],n[l−1])×(n[l−1],m)+(n[l],1)(n^{[l]}, m) = (n^{[l]}, n^{[l-1]}) \times (n^{[l-1]}, m) + (n^{[l]}, 1)(n[l],m)=(n[l],n[l−1])×(n[l−1],m)+(n[l],1)

A[l]=g[l](Z[l])A^{[l]} = g^{[l]}(Z^{[l]}) A[l]=g[l](Z[l])

Shapes: (n[l],m)=g((n[l],m))(n^{[l]}, m) = g((n^{[l]}, m))(n[l],m)=g((n[l],m))

where:

  • W[l]W^{[l]}W[l] is the weight matrix for layer lll with shape (n[l],n[l−1])(n^{[l]}, n^{[l-1]})(n[l],n[l−1])
  • b[l]b^{[l]}b[l] is the bias vector for layer lll with shape (n[l],1)(n^{[l]}, 1)(n[l],1)
  • g[l]g^{[l]}g[l] is the activation function for layer lll
  • A[0]=XA^{[0]} = XA[0]=X (input features) with shape (n[0],m)(n^{[0]}, m)(n[0],m)

    Zraw[l]=W[l]A[l−1]+b[l]Z_{\text{raw}}^{[l]} = W^{[l]} A^{[l-1]} + b^{[l]} Zraw[l]​=W[l]A[l−1]+b[l]

  5. Activation:

    Shapes: (n[l],m)=g((n[l],m))(n^{[l]}, m) = g((n^{[l]}, m))(n[l],m)=g((n[l],m))

    Shapes: (n[l],1)=scalar×(n[l],1)+scalar×(n[l],1)(n^{[l]}, 1) = \text{scalar} \times (n^{[l]}, 1) + \text{scalar} \times (n^{[l]}, 1)(n[l],1)=scalar×(n[l],1)+scalar×(n[l],1)

    Znorm[l]=Zraw[l]−running_mean[l]running_var[l]+ϵZ_{\text{norm}}^{[l]} = \frac{Z_{\text{raw}}^{[l]} - \text{running\_mean}^{[l]}}{\sqrt{\text{running\_var}^{[l]} + \epsilon}} Znorm[l]​=running_var[l]+ϵ​Zraw[l]​−running_mean[l]​

Dropout

Applied to hidden layers during training:

A[l]=A[l]⊙D[l]keep_probA^{[l]} = A^{[l]} \odot \frac{D^{[l]}}{\text{keep\_prob}} A[l]=A[l]⊙keep_probD[l]​

where D[l]D^{[l]}D[l] is a binary mask with probability keep_prob of being 1.
